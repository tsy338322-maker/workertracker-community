# WorkerTracker Community

## Repository Purpose

This repository is intended for demonstration, portfolio, and community reference purposes.

Some production features, deployment configurations, and operational components are intentionally excluded from the public version.

## Features

* Workforce Management
* User Management
* Access Control
* Ownership Isolation
* Resource Tracking
* Android Application Support

## Technology

* JavaScript
* Node.js
* HTML/CSS
* Android (Capacitor)

## Community Edition

This repository contains the community showcase version of WorkerTracker.

Production data, deployment secrets, environment-specific configurations, and operational resources are not included in this public repository.

## Status

Active development continues in a separate private production repository.
